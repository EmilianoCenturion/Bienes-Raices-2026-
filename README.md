# Bienes Raíces 2026

Proyecto web inmobiliario desarrollado como práctica profesional.

## 🚀 Tecnologías utilizadas
- HTML5
- SCSS (Sass)
- Gulp
- JavaScript
- Node.js

## 📦 Instalación

Clonar el repositorio:

git clone https://github.com/EmilianoCenturion/Bienes-Raices-2026-.git

Instalar dependencias:

npm install

Ejecutar entorno de desarrollo:

npm run dev

## 📁 Estructura del proyecto

- src → Archivos fuente (SCSS, JS, imágenes)
- build → Archivos compilados
- gulpfile.js → Automatización de tareas
